# 📚 Book Shopping MVC

A full-featured **Book Store web application** built with **ASP.NET Core MVC**, **Razor Views**, and **Entity Framework Core**. This project was created to practice modern ASP.NET Core development while implementing real-world e-commerce features such as authentication, shopping cart, order management, inventory management, and reporting.

---

## ✨ Features

### 👤 User Features

- Register with Email & Password
- Login with Email & Password
- Google Authentication (OAuth)
- Email Confirmation after registration
- Forgot Password & Password Reset via Email
- Edit Profile Information
- Browse Books
- Search books by title
- Filter books by category
- View book details
- Add books to shopping cart
- Update quantity for each cart item
- Remove items from cart
- Checkout

---

### 🛒 Shopping Features

- Shopping Cart
- Quantity management
- Stock validation
- Order creation
- Checkout process

---

### 📚 Book Management (Admin)

- Add/Edit/Delete books
- Upload book images
- Manage book categories
- Manage stock separately
- Stock availability validation

---

### 📦 Order Management (Admin)

- View all orders
- Update order status
- Update payment status
- Track customer orders

---

### 📊 Reports

- Top Selling Books Report
- Filter reports by date

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/BookShoppingMVC.git
```

```bash
cd BookShoppingMVC
```

Update the connection string in:

```
appsettings.json
```

Apply migrations:

```bash
dotnet ef database update
```

Run the project:

```bash
dotnet run
```
